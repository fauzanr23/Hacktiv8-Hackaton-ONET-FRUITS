# Hacktiv8-Hackaton---Puzzle-Game
This repository contains assets of a web matching puzzle game

//game: onad (puzzle) --> gambar (hewan, buah, warna/bentuk)

//CRUD
onad:
C: nama, score
R: tampilan score
U: score ++
D: menghilangkan item


//Catatan Tambahan
sementara 1 level (bisa bertambah lvl dengan pengurangan waktu & penambahan gambar)

pakai Math.random(), database local (array of object)


//Pembagian tugas:

bikin tampilan game (front-end): tampilan awal, alert gagal dan berhasil, tampilan waktu, asset game (start button, kartu,  gambar, music)
 
bikin logic: random kartu, score, penghilangan kartu (gambar match), kartu kembali kosong (gambar berbeda),  input nama, game start (waktu mulai berkurang), akhir game (game over/berhasil)

Ozy: penghilangan kartu (gambar match), score
Indra:  input nama (logic nyimpan ke string), tampilan awal (enter name, tombol start game)
Conan: game start (waktu mulai berkurang), tampilan waktu
Fauzan: random kartu, akhir game (game over/berhasil), github, asset (gambar, music)

aset gambar: https://www.flaticon.com/search?word=fruits
